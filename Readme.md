# MИС БАРС — Автоматическое назначение анализов

Расширение для Яндекс.Браузера (на базе Chromium), которое автоматически проставляет галки напротив стандартных наборов лабораторных анализов в медицинской информационной системе **МИС БАРС**.

## Возможности

- ✅ Автоматическое проставление галочек напротив нужных анализов
- 🔎 Автоматический переход в раздел «Все исследования» перед заполнением
- 📁 Готовые профили для различных клинических ситуаций
- ⚙️ Возможность создавать, редактировать и удалять профили прямо в расширении
- 🔔 Автоматическая попытка выставить отображение всех записей (150)
- 🔒 Все данные хранятся локально — ничего не передаётся на внешние серверы

## Установка

1. Скачайте или склонируйте репозиторий:
   ```bash
   git clone https://github.com/ваш_username/auto-form-filler.git
Откройте Яндекс.Браузер и перейдите по адресу:

text
browser://extensions
Включите «Режим разработчика» (Developer mode)

Нажмите «Загрузить распакованное» (Load unpacked)

Выберите папку с расширением

Использование
Откройте карту пациента в МИС БАРС и перейдите к разделу назначения анализов

Расширение сначала попробует открыть раздел «Все исследования», затем автоматически установить отображение всех анализов — 150 записей.

Если автоматическое переключение не сработало, установите отображение вручную:

Нажмите на цифру в правом нижнем углу таблицы (20, 50 и т.д.)

В поле ввода введите 150 и нажмите Enter

Дождитесь загрузки всех анализов

Нажмите на иконку расширения на панели инструментов

Выберите нужный профиль из выпадающего списка

Нажмите «Заполнить форму»

Галки напротив нужных анализов проставятся автоматически. Выбранный профиль запоминается для следующего раза.

Создание и редактирование профилей
Нажмите кнопку настроек рядом с заголовком «Назначение анализов». В меню можно добавить новый профиль, выбрать иконку, найти нужные исследования по названию или коду, отметить их галочками и сохранить назначение.

Изменения профилей хранятся локально в браузере.
Как узнать item_value анализа
Откройте страницу с анализами в МИС БАРС

Нажмите F12 → выберите инструмент выбора элемента (прицел)

Кликните по чек-боксу нужного анализа

В коде найдите атрибут item_value="..."
