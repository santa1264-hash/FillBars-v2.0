# МИС БАРС - автоматическое назначение анализов

Расширение для Яндекс.Браузера и других Chromium-браузеров. Оно помогает быстро назначать готовые наборы лабораторных исследований в МИС БАРС: открывает раздел со всеми исследованиями, пытается показать полный список на 150 записей и последовательно отмечает нужные чекбоксы так, чтобы БАРС успевал перенести выбранные исследования в нижний блок формы.

По настройке расширение может остановиться сразу после выбора анализов или продолжить сценарий дальше: нажать **Назначить**, открыть подбор времени записи на услугу, выбрать кабинет **Лаборатория Амурск** для строк записи и проставить галочки **Срочно**. Финальную кнопку записи расширение не нажимает.

Текущая версия интерфейса: **3.0**.

## Что умеет расширение

- Выбирает готовый профиль назначений из выпадающего списка.
- Перед заполнением пытается нажать кнопку **Все исследования** в форме БАРС.
- Автоматически пытается выставить отображение **150 записей** на странице.
- Последовательно кликает нужные чекбоксы и дает БАРС время обработать каждое выбранное исследование.
- Показывает результат заполнения и предупреждает, если на странице найдено слишком мало исследований.
- Запоминает последний выбранный профиль.
- Позволяет выбрать этап выполнения сценария через слайдер в настройках.
- В полном режиме после выбора анализов нажимает **Назначить**, выбирает кабинет **Лаборатория Амурск** и ставит **Срочно**.
- Позволяет создавать, редактировать и удалять профили прямо в расширении.
- Считывает список исследований с текущей страницы БАРС для удобного выбора в редакторе.
- Хранит пользовательские профили и каталог исследований локально в браузере.

## Этапы выполнения

В настройках есть слайдер **Этап выполнения**:

- **Только анализы** - расширение выбирает исследования по профилю и останавливается. Кнопка **Назначить** не нажимается.
- **Кабинеты и срочно** - расширение выбирает исследования, нажимает **Назначить**, в окне подбора времени выбирает **Лаборатория Амурск** для строк записи и ставит галочки **Срочно**.

Выбранный режим сохраняется локально и используется при следующих запусках.

## Как работает заполнение анализов

1. Пользователь открывает форму назначения анализов в МИС БАРС.
2. В расширении выбирает профиль и нажимает **Заполнить форму**.
3. Расширение запускает скрипт на активной вкладке БАРС.
4. Скрипт ищет и нажимает **Все исследования**.
5. Затем пытается установить размер списка на **150 записей** через select, поле ввода или кликабельный контрол в нижней части таблицы.
6. Если исследований все равно меньше 100, появляется предупреждение, что 150 записей нужно выставить вручную.
7. Для каждого исследования из выбранного профиля расширение находит чекбокс по `item_value`, прокручивает строку в видимую область и кликает по чекбоксу как пользователь.
8. Если чекбокс уже был отмечен, расширение сначала снимает отметку и ставит ее заново, чтобы БАРС обновил нижний список выбранных анализов.

Важно: БАРС может загружать форму медленно. После открытия формы нужно дождаться загрузки таблицы и не нажимать кнопки несколько раз подряд.

## Как работает полный режим

Если выбран этап **Кабинеты и срочно**, после отметки исследований расширение:

1. Нажимает кнопку **Назначить**.
2. Ждет окно **Подбор времени записи на услугу**.
3. Находит строки подбора времени и кнопки выбора кабинета.
4. Для каждой строки открывает справочник **Кабинеты**.
5. Выбирает строку **Лаборатория Амурск** двойным кликом или подтверждением через окно справочника.
6. Переходит к следующей строке, если окно справочника закрылось и выбор применился.
7. После выбора кабинетов ставит галочки **Срочно**.

Кнопку финального подтверждения записи расширение не нажимает: пользователь сам проверяет результат и продолжает работу в БАРС.

## Профили назначений

В чистой установке с GitHub расширение содержит два профиля:

- 🧪 ОНМК при поступлении
- 💊 Ежедневный анализ РАО

Коагулограмма и старые хирургические шаблоны не входят в рабочий список чистой установки.

После установки список профилей можно менять в настройках. Пользовательские изменения сохраняются локально в браузере и могут отличаться от исходного набора из репозитория.

## Редактирование профилей

Кнопка настроек находится справа от заголовка **Назначение анализов**. Она открывает панель **Профили назначений**.

В панели можно:

- выбрать этап выполнения сценария;
- открыть существующий профиль на редактирование;
- удалить профиль;
- добавить новое назначение;
- выбрать иконку назначения;
- задать название;
- найти исследование по названию или коду;
- отметить нужные исследования галочками;
- сохранить новый или измененный профиль.

Кнопка **Обновить список** в редакторе считывает исследования с открытой страницы БАРС. При чтении расширение тоже пытается открыть **Все исследования** и выставить **150 записей**, затем сохраняет найденные названия локально. Технические коды, материалы вроде "сыворотка крови" и служебные значения не показываются в списке выбора.

Пользовательские профили сохраняются в `localStorage` браузера. Если удалить встроенный профиль, расширение запомнит это локально и не будет показывать его снова, пока локальные данные расширения не будут очищены.

## Установка

1. Скачайте или склонируйте репозиторий:

   ```bash
   git clone git@github.com:santa1264-hash/FillBars-v2.0.git
   ```

2. Откройте в Яндекс.Браузере страницу:

   ```text
   browser://extensions
   ```

3. Включите режим разработчика.
4. Нажмите **Загрузить распакованное расширение**.
5. Выберите папку проекта с файлами `manifest.json`, `popup.html` и `popup.js`.

## Использование

1. Откройте в МИС БАРС форму назначения анализов.
2. Дождитесь загрузки таблицы исследований.
3. Откройте расширение на панели браузера.
4. При необходимости откройте настройки и выберите этап выполнения: **Только анализы** или **Кабинеты и срочно**.
5. Выберите нужный профиль назначений.
6. Нажмите **Заполнить форму**.
7. Дождитесь завершения работы расширения и проверьте выбранные анализы, кабинеты и срочность в форме БАРС.

Если расширение сообщает, что найдено мало анализов, вручную нажмите на число записей в правом нижнем углу таблицы, введите `150`, нажмите Enter, дождитесь загрузки и повторите заполнение.

## Хранение данных

Расширение ничего не отправляет на внешние серверы. Локально в браузере хранятся:

- пользовательские профили;
- список удаленных встроенных профилей;
- каталог считанных исследований;
- последний выбранный профиль.
- выбранный этап выполнения сценария.

## Ограничения

- Расширение рассчитано на текущую форму назначений МИС БАРС и чекбоксы с `item_value`.
- Если интерфейс БАРС изменится, автоматическое открытие **Все исследования** или переключение на 150 записей может потребовать доработки.
- Расширение помогает автоматизировать рутинные клики, но не принимает медицинские решения.
